# Quick Reference Card - 3 AQI Pages

## 🔍 AQI Sources - "What's causing pollution?"

**Elevator Pitch**: "Identifies pollution sources and their contribution percentages"

**Show Client**:
- Source breakdown: Traffic 35%, Industrial 28%, Construction 18%
- Pollutant fingerprints: NO₂ = traffic, SO₂ = industrial
- Evidence chain: Station data → Source identification

**Value**: Target the biggest contributors, not waste resources on minor sources

**Example**: "Instead of blanket restrictions, focus on traffic (35%) and industry (28%)"

---

## ⚙️ AQI Agents - "Is the AI working?"

**Elevator Pitch**: "System health dashboard - ensures your AI infrastructure is running 24/7"

**Show Client**:
- 8 AI agents (Ingestion, Forecasting, Alerts, etc.)
- System metrics: 83% success rate, 38s avg execution
- Real-time status: Running, Completed, Failed
- Execution logs with error tracking

**Value**: Catch system failures before users notice, ensure reliability

**Example**: "Alert Agent failed at 6am - we fixed it before morning commute notifications"

---

## 📈 AQI Mitigation - "What worked and by how much?"

**Elevator Pitch**: "Tracks pollution reduction actions and proves ROI"

**Show Client**:
- Action: Traffic Rerouting
- Before: 165 AQI (Unhealthy)
- After: 142 AQI (Moderate)
- Reduction: -23 points
- Effectiveness: 85%
- Total impact: -72 AQI across all actions

**Value**: Justify budgets, prove interventions work, optimize spending

**Example**: "Our $50K traffic intervention reduced pollution by 23 points - 85% effective!"

## Client Objection Handling

**"Why three pages instead of one?"**
→ Different audiences: Environmental managers need Sources/Mitigation, IT needs Agents

**"Seems complicated"**
→ It's actually simpler: Find problem (Sources) → Ensure tools work (Agents) → Measure results (Mitigation)

**"Can't afford three features"**
→ They work together: Sources finds the problem, Mitigation proves the ROI. Agents ensures uptime (no downtime = no lost revenue)

**"We just need basic monitoring"**
→ Basic monitoring tells you pollution is high. Our system tells you WHY (Sources), ensures reliability (Agents), and proves actions work (Mitigation)
