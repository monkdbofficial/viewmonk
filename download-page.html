Workflow

From download to insights in 4 steps

No cloud sign-up, no configuration sprawl. Download and run.

1
Download & Extract
Grab LITE (5 MB) or FULL (366 MB). Extract ZIP or TAR.GZ — no installer, no admin rights needed.
2
Install & Launch
Run ./start.sh or start.bat. LITE auto-installs deps once; FULL runs immediately offline.
3
Connect to monkdb
Point ViewMonk at your monkdb instance on port 4200 — local, remote, Kubernetes, or cloud-hosted.
4
Explore & Build
Query with SQL, build dashboards, run geospatial analysis, search vectors, and manage users — all in one UI.
Quick Start

Up and running in minutes

Follow these steps to deploy ViewMonk (Workbench) locally or on your server.

System Requirements

Verify these prerequisites before installing.

RuntimeNode.js 18+
OSWin · macOS · Linux
BrowserChrome / Firefox / Safari / Edge
Databasemonkdb (port 4200)
LITE networkInternet on first run
FULL disk~600 MB extracted
1

Verify Node.js

Node.js 18 or later is required. Check your installed version:

~ bash
$ node --version # Should output v18.0.0 or higher # Download from nodejs.org if needed
2

Download & Extract

Download your preferred edition above, then extract the archive:

~ bash
# Linux / macOS (LITE) $ tar -xzf monkdb-workbench-standalone-lite.tar.gz $ cd monkdb-workbench-standalone-lite # Windows — right-click ZIP → "Extract All"
3

Install dependencies (LITE only)

Skip if you downloaded FULL — it ships with all dependencies pre-bundled.

~ bash
$ npm install # Downloads ~360 MB once — subsequent launches are offline
4

Launch the server

Linux / macOS:

~ bash
$ ./start.sh # Or directly: node server.js

Windows:

~ cmd
Double-click start.bat — or in terminal — > node server.js
5

Open in your browser

Navigate to ViewMonk and connect to your monkdb instance:

~ browser
http://localhost:3000 # Custom port: $ PORT=8080 node server.js
